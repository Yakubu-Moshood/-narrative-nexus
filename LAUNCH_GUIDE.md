## ðŸ“ž Support & Issues

### Common Issues & Solutions

**Issue**: App takes too long to load
- **Solution**: Streamlit Sharing can be slow. Refresh page or wait 10 seconds.

**Issue**: Upload fails with error
- **Solution**: Ensure file is TXT or CSV, max 50MB, has proper headers.

**Issue**: Stories don't match my data
- **Solution**: Try uploading your CSV for personalized insights.

**Issue**: Mobile view is broken
- **Solution**: Try landscape mode or access from desktop for full experience.

### Report a Bug
- Create an issue on GitHub
- Include: What you were doing, what happened, expected result
- Attach screenshots if possible
